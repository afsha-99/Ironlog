# Iron Log 💪

Gym Tracker Web App — läuft direkt im iPhone Browser ohne Installation.

## 🚀 GitHub Pages Setup (5 Minuten)

1. **Neues GitHub Repository erstellen** → Name z.B. `ironlog`
2. **Diese Dateien hochladen:**
   - `index.html`
   - `manifest.json`
   - `sw.js`
   - `icon.png`
3. **GitHub Pages aktivieren:**
   - Repository → Settings → Pages
   - Source: `main` Branch → `/ (root)`
   - Save
4. **Deine App URL:** `https://DEIN-USERNAME.github.io/ironlog`

## 📱 Auf iPhone installieren

1. URL in **Safari** öffnen
2. Teilen-Button → **"Zum Home-Bildschirm"**
3. Fertig — App-Icon auf dem Home-Bildschirm!

## 💾 Daten

- Daten werden in `localStorage` gespeichert (bleibt erhalten)
- Neue Nutzer starten mit einer vorgefüllten Exercise-Liste
- Nutzer können diese Exercises in der App löschen, bearbeiten oder eigene hinzufügen
- Backup: In der App → Backup Tab → **"Download Backup File"**
- Restore: Backup-Datei laden wenn nötig

## 🔄 App aktualisieren

Einfach `index.html` in GitHub ersetzen → App updated sich automatisch.
